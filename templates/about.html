Project Overview & Insights

Machine Learning Models

  • Linear Regression: Linear Regression is a basic algorithm used to model the relationship between a dependent variable and one or more independent variables. It assumes a linear relationship, which can limit its flexibility but offers high interpretability and fast computation.
  • Random Forest: Random Forest is an ensemble model that builds multiple decision trees and averages their results to improve accuracy and reduce overfitting. It handles non-linear relationships well, but can be less interpretable and slower than simpler models.
  • Gradient Boosting: Gradient Boosting builds trees sequentially, where each new tree corrects errors made by previous ones. This method often delivers high accuracy, especially for structured/tabular data. However, its sensitive to hyperparameters and more prone to overfitting than Random Forest.

Chosen Model: Gradient Boosting Regressor for CLV prediction due to best performance on hold-out set (lowest MAE).

Dashboard Graph Insights

  • Weekly Spend Over Time: Tracks overall spending trends and seasonality.
  • Top Departments by Spend: Identifies which categories drive revenue.
  • Product Combinations: Shows top pairs bought together to support cross-sell strategies.
  • Cross-Sell Targets: Given a product, shows likely pairings using attachment probability.
  • Top 20 CLV Customers: Prioritize these high-value shoppers for loyalty rewards.
  • Top 20 Churn Risks: At-risk customers to re-engage through targeted outreach.
